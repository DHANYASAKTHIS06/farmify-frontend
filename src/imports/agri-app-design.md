Design a modern mobile app UI for an agriculture marketplace platform called “AGRI” that connects organic farmers and customers.

🎨 Style

Clean, minimal, nature-inspired design

Green and earthy color palette

Rounded cards and soft shadows

Simple icons and large readable typography

Support for Tamil, English, and Hindi

📱 Screens to Design
1️⃣ Splash / Intro Screen

App name: AGRI

Tagline: “Connecting Organic Farms to People”

Voice intro option (speaker button)

Language selection: Tamil | English | Hindi

2️⃣ Farmer Registration Screen

Fields:

Farmer Name

Phone Number

Location (with map picker)

Address

Voice input button option

CTA: Register

3️⃣ Farmer Product Upload Screen

Fields:

Upload Photo

Product Name

Quantity

Organic toggle (Yes/No)

Breed (optional)

Expiry Date (after this date product auto-hides)

Include:

Notification reminder before expiry

CTA: Add Product

4️⃣ Customer Home Screen

Sections:

Nearby Organic Farms (map view option)

Search by Fruit / Vegetable

AI Match Recommendations (based on user needs)

Pick Your Own Fruit Option

Visit Booking / Call Button

Delivery Option (e.g., Egg, Milk etc.)

5️⃣ Farm Details Page

Farm photos

Government Organic Certificate badge (verified mark)

Certificate verification status

Ratings & Reviews

Report Farm button

Call / Book Visit button

Navigation to Farm

6️⃣ AI Assistant Screen

Chat interface

Mic button for voice interaction

Explains schemes and matches:

Farmer problems → Shows matching government schemes

School requirements → Matches available farmers

7️⃣ Profile Screen

Display numbers (orders, visits, bookings)

History (All activity)

Notifications

Settings (Language change)

⚙️ Backend Features to Reflect in UI

Government certificate verification system

Fake certificate detection indicator

Report customer / Report farmer feature

Rating system

Expiry-based auto product removal

Push notifications
